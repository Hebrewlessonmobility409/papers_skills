## 🛠️ Frequently asked questions

### Does this software modify my original files?
No. The application creates a link to your original files. It reads the files to provide search and organization features, but it leaves your base documents in their original folders.

### Can I share my document library?
You can export your tag list as a CSV file. This allows you to import your settings into another computer.

### How do I update to the latest version?
Download the new version from the link listed above. The installer replaces the old files automatically and keeps your existing data intact.

### Is my data private?
All data stays on your local machine. The software does not send your documents or tags to external servers. Your research stays inside your walls.

## 💡 Best practices for new users

Start by choosing one folder that contains your most important research. Point the application to this folder during the start-up process. Let the system index the files. Once the indexing completes, create your first few tags. Apply these tags to your papers. This structure helps you find specific information as your library grows.

Do not try to move all your folders at once. Start small. A library of fifty files is easier to manage than a library of five thousand. Once you understand the tagging interface, add your larger folders. This method ensures accuracy and keeps your index clean. 

If the search does not display a file immediately, check the folder path in the settings menu. Ensure the application has read access to the directory. Refresh the index if you add new files to the source folders outside of the application interface.

## 🆘 Troubleshooting common issues

If you encounter a problem, check these items first:

* Application does not launch: Reinstall the software using the administrator account on your Windows machine.
* Tags disappear: Check your storage settings. Ensure the application has permission to write data to your local user folder.
* Slow search speeds: If you have a very large library, the initial index takes time. Close other programs while the index builds. Once finished, performance becomes stable.
* Link broken: Ensure your firewall allows the application to check for updates. If you still have trouble, visit the GitHub release page to download the latest manual installer.
